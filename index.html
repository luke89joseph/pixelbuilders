What's binary?

Binary is how computers process information!


When we provide computers code, the computer takes our code and converts it as such:

Code Language (Java, HTML, etc.) ->

Assembly Language (Depends on the OS/Operating System, ie. Mac and Windows) ->

Binary!!



Binary uses 0s and 1s to express data, like so:

1101110101011010101101 (this is a byte)


While only using 0s and 1s sounds VERY limiting, we can actually express ALOT!

It's important to note that each byte (single expression) represents a singular character, like a letter (a, b, c) or number (1, 2, 3).

10110110101101 <- byte

Each 0 and 1 in a byte is called a bit

1 <- bit

What's binary?

Binary is how computers process information!


When we provide computers code, the computer takes our code and converts it as such:

Code Language (Java, HTML, etc.) ->

Assembly Language (Depends on the OS/Operating System, ie. Mac and Windows) ->

Binary!!



Binary uses 0s and 1s to express data, like so:

10110101 (this is a byte)


While only using 0s and 1s sounds VERY limiting, we can actually express ALOT!

It's important to note that each byte (single expression) represents a singular character, like a letter (a, b, c) or number (1, 2, 3).

10010110 <- byte

Each 0 and 1 in a byte is called a bit

1 <- bit
(We can have 8 bits in a byte)


So, how are we able to express data through a series of 0s and 1s?

Imagine all the different combinations of 0s and 1s you could make:

For an 8 digit sequence (1-byte), we can make 256 different combinations

For an 32 digit sequence (4-bytes), we can make 4,294,967,296 different combinations

---

As you can see, just the addition of just 3-bytes is HUGE!
The amount of combinations we can make exponentially increases with each byte we add.

Here, we add the same amount each time: Normal: 1, 2, 3, 4


Here, we double the amount we add each time: Exponential: 1, 2, 4, 16



Given all these available combinations, computers assign each combination to a certain character

For example:

A - 01000001

B - 01000010

C - 01000011

1 - 00110001

2 - 00110010
